웹 스크래핑 -->
모든 웹은 HTML로 구성되어 있으므로, HTML의 규칙을 파악한다면
얼마든지 HTML에서 필요한 정보를 가져옴. 이 과정을 말함.

** 정규 표현식: 문자를 표현하는 공식으로, 특정 규칙이 있는 문자열 집합 추출 시 사용

XML(eXtensible Markup Language): 확장 마크업 언어
-> 데이터의 구조와 의미를 설명하는 태그를 사용해 어떤 데이터의 속성과 값을 표현하는 언어
-> 시작 태그와 종료 태그 사이에 어떤 값이 있고, 그 값은 태그의 이름으로 만들어진 속성에 대한 값이 됨

JSON: XML보다 데이터 용량이 적고 코드로의 전환이 쉬운 XML의 대체제로써 가장 많이 활용됨
-> 키-값 쌍으로 구성되어, 파이썬 딕셔너리형과 매우 비슷
